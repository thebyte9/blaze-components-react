# blaze-components-react

Storybook: https://blaze-components-react.thisisblaze.com

## Getting Started

### Running storybook

`yarn storybook`

### Development

#### Lint files

`yarn lint`

#### Run tests

`yarn test`

#### Build library

`yarn build`

#### Build storybook as a static web application

`yarn build-storybook`

#### Run server

`npx http-server ./build`

### Publish

- yarn build && yarn build-storybook
- commit changes
- lerna publish
