
## Description

The VideoContainer component  is a simple wrapper to make the video responsive

## Usage


```js
<VideoContainer
    src="https://www.youtube.com/embed/sSJiZLkfsnw"
    title="lorem ipsum"
    frameborder="0"
  />
```

## API

##### VideoContainer can receive a number of `props` as follow:


| NAME   | TYPE | DEFAULT | 
| :---  | :---:  | :---: |
| src | string | empty | 
| title | String | empty | 
| frameborder | string | '0' |
